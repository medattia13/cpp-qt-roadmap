# Week 1: Modern C++ Foundations

## Day 2 – Review

### Topics Covered
- Variables
- References
- `const`
- Functions
- Function overloading

### Exercises

Create functions for the following:

1. Factorial
2. Fibonacci
3. Greatest Common Divisor (GCD)
4. Least Common Multiple (LCM)
5. Prime number checking

---

## Day 3 – Study

### Topics Covered
- References
- Const correctness
- RAII (Resource Acquisition Is Initialization)
- Smart pointers
- Move semantics

### Exercises

Create classes that demonstrate:

1. Reference members
2. Const member functions
3. RAII for resource management
4. Smart pointers (`std::unique_ptr`, `std::shared_ptr`)
5. Move constructor
6. Move assignment operator

---

## Learning Goals

By the end of Week 1, you should be able to:

- Understand variables and references.
- Use `const` correctly.
- Write and overload functions.
- Implement common mathematical algorithms.
- Apply RAII principles.
- Use smart pointers safely.
- Understand and implement move semantics.
